Инструкция по сборке релиза

1. Запустить в QtCreator программу в конфигурации Release
2. Получившийся exe файл скопировать в c:\1
3. В c:\1 скопировать все *.qml, *.ico из папки imp
4. Запустить командную строку с окружением MinGW 64
5. В командной строке запустить утилиту с параметром: windeployqt c:\1
6. Перенести всю папку 1 в build (см. раздел Source в iss файле)
7. Запускай iss файл
8. Результат будет лежать в соответствии с указанием в OutputDir файла iss
