Запуск и закрытие приложения
============================

|

Для работы в приложении **ИМП** необходимо запустить исполняемый файл :bolditalic:`imp.exe`.

При запуске приложения производится поиск доступных измерителей, которые затем отобразятся в главном окне.
Если приложение запускалось ранее, то по окончании поиска измерителей оно автоматически восстановит предыдущий режим работы.

Закрытие приложения осуществляется стандартными средствами *Windows*.

.. tip:: Для автоматического восстановления текущего режима работы приложения при следующем запуске достаточно закрыть только главное окно приложения.

 При выходе из приложения все дочерние окна будут закрыты автоматически.

.. attention:: Окна индикаторов, закрытые пользователем вручную, при следующем запуске приложения восстановлены не будут.
